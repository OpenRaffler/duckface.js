Duckface
==================

Javascript interfaces by duck typing!

.. contents:: Table of Contents

Installation
-----

Use `Bower`_ to install **Duckface** into your project::

    $ bower install duckface

Then use `RequireJS`_ to use the library::

    require(['duckface']);
